HTML 头部

  • 文件的标题
    <title>标题就是网页标签上的文字</title>
  • 所有链接一个目标
    <base> 标签为页面上的所有链接规定默认地址或默认目标（target）
  1. <head>
  2. <base href="http://www.w3school.com.cn/images/" />
  3. <base target="_blank" />
  4. </head>
  • 使用<meta>元素来描述文档。
    • 元数据（metadata）是关于数据的信息。
    • 标签提供关于 HTML 文档的元数据。
    • 元数据不会显示在页面上，但是对于机器是可读的。
    • 典型的情况是，meta 元素被用于规定页面的描述、关键词、文档的作者、最后修改时间以及其他元数据。
  1. //example:
  2. <meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
  3. <meta name="author" content="w3school.com.cn">
  4. <meta name="description" content="Free Web tutorials on HTML" />
  • 重定向用户(跳转网页)
  1. <meta http-equiv="Content-Type" content="text/html; charset=gb2312" />
  2. <meta http-equiv="Refresh" content="5;url=http://www.w3school.com.cn" />

HTML 其他标签

  1. HTML实体:
  2. HTML 中的常用字符实体是不间断空格 "&nbsp;"
  3. 浏览器总是会截短 HTML 页面中的空格。如果您在文本中写 10 个空格，在显示该页面之前，浏览器会删除它们中的 9 个。如需在页面中增加空格的数量，您需要使用 &nbsp; 字符实体。

比如百度贴吧的 回复栏

  1. 换行符是 </br>
  2. 在HTML文档中，回车并不能换行，需要使用换行符，但部分标签是单独一行的。
  3. 如<p> 等等
  1. <pre></pre>标签不改变其内容的排版方式
  2. 需要用到一段代码时，会使用到。
  3. <pre>
  4. int main(){
  5.     return 0;
  6. }
  7. </pre>
  1. <address> 标签定义文档或文章的作者/拥有者的联系信息
  2. 关于 mailto 标签： 当我们点击了example@phplamp.com链接后浏览器会为我们自动选择本地的默认邮件软件，并将mailto标签后example@phplamp.com邮件地址放到邮件软件的发送地址中，少去了我们发送邮件时输入发送地址的烦扰，同时也可以避免输错邮件地址。
  3. <address>
  4.     <a href="mailto:service@s3school.com.cn">发送邮件</a>
  5. </address>

虚拟机的使用

虚拟机有什么用？
学习一些软件又不想在本机上操作时，使用虚拟机在本机的硬盘中虚拟一台电脑，虚拟出来的电脑和真实机差别不大，以供学习使用。

主流的虚拟机有
+ Oracle VM VirtualBox (free)
+ VMware Workstation (not free)

如有兴趣可自行查找资料。要使用虚拟机，需要系统镜像文件来安装操作系统。
